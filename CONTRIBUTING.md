# Contributing

Preferred contributor workflow:

1. Put your skill in a public GitHub repo (a folder containing `SKILL.md`).
2. Use the registry site `/import` page to generate an import issue.
3. A maintainer applies label `import-approved` to trigger an automated PR.

Start here:

- `docs/contributing.md`
- `docs/importer.md`
