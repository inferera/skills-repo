# Hello World Skill

A simple example skill that demonstrates the basic structure and usage of AI agent skills.

## Overview

This is a beginner-friendly skill that shows you how to create and structure your first AI agent skill. Perfect for learning the basics!

## Features

- ✅ Simple and easy to understand
- ✅ Well-documented structure
- ✅ Ready to use examples
- ✅ Best practices included

## Installation

```bash
# Install the skill to your agent's skills directory
cp -r hello-world ~/.claude/skills/
```

## Usage

Once installed, the agent can use this skill automatically. Here's what it does:

1. Demonstrates basic skill structure
2. Shows how to document skills
3. Provides examples for beginners

## Example

```javascript
// Example usage
console.log("Hello from the AI agent skill!");
```

## Configuration

No configuration needed! This skill works out of the box.

## Requirements

- Node.js 18+
- Claude agent
